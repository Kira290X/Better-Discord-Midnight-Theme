Midnight is a sleek BetterDiscord theme featuring a deep black color palette for a modern, minimalist look. Perfect for users who prefer a clean, dark interface thatâ€™s easy on the eyes during extended use. Fully compatible with the latest BetterDiscord version and easily customizable.

## Installation

1. Ensure BetterDiscord is installed.
2. Go to BetterDiscord settings in Discord.
3. Click on **"Themes"** and then **"Open Theme Folder"**.
4. Place the downloaded CSS file into this folder.
5. Return to Discord and enable the theme in BetterDiscord settings.
